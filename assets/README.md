# Project Assets

This directory contains showcase assets for the Self-Healing AI CI/CD Agent.

## Contents

- `dashboard.png` — Benchmark Dashboard screenshot (add your own)
- `analysis.png` — Analysis view screenshot (add your own)
- `review.png` — Multi-agent review panel screenshot (add your own)
- `pr-automation.png` — PR automation view screenshot (add your own)

## Adding Screenshots

1. Run the application locally (see [README](../README.md))
2. Navigate to the relevant page
3. Take a screenshot
4. Save the PNG file in this directory
5. Reference it in the README using `![alt](assets/filename.png)`
